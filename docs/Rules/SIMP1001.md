# SIMP1001 - Usage of the Greater Than Sign ('>')

## Summary

Detects all usages of the greater than sign (`>`) in C# code.

## Rationale
Don't use the greater than sign in programming


## Non-compliant code

```csharp
if (a > b)
{
    // ...
}
```

## Compliant code

```csharp
if (b < a)
{
    // ...
}
```
